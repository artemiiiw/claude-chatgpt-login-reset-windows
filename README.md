```powershell

```
- macOS/Linux:
```bash
curl -L "https://github.com/artemiiiw/claude-chatgpt-login-reset/raw/master/scripts/macos/claude-chatgpt-login-reset-macos.sh" -o ./claude-chatgpt-login-reset-macos.sh

```
